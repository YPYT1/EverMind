# Security

EverMind is local-first, but local services still need care.

- Bind EverOS to `127.0.0.1` unless you provide your own authentication gateway.
- Never commit `.env`.
- Do not store API keys, tokens, passwords, cookies, private keys, or session credentials in memory.
- Treat Basic Memory as reviewed project knowledge, not a secret store.
- Review Basic Memory candidates before committing official notes.
