# EverMind Agent Instructions for Claude Code

EverMind provides local project memory through EverOS, MCP, and reviewed Basic Memory notes.

## Workflow

- Start coding sessions by checking `briefing` and relevant `recall` results.
- Use Basic Memory notes as durable project knowledge, then verify important facts in files.
- For unfamiliar repositories, use codebase-memory to inspect architecture and impact.
- After meaningful changes, propose a Basic Memory candidate with evidence.
- Do not write official long-term notes without explicit user confirmation.

## Safety

Never save API keys, tokens, passwords, cookies, private keys, or session credentials.
