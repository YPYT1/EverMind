# EverMind Agent Instructions for Claude Code

EverMind provides local project memory through EverOS, MCP, and reviewed EverMind Archive notes.

## Workflow

- Start coding sessions by checking `briefing` and relevant `recall` results.
- Use EverMind Archive notes as durable project knowledge, then verify important facts in files.
- For unfamiliar repositories, use evermind-code-graph to inspect architecture and impact.
- After meaningful changes, propose a EverMind Archive candidate with evidence.
- Do not write official long-term notes without explicit user confirmation.

## Safety

Never save API keys, tokens, passwords, cookies, private keys, or session credentials.
